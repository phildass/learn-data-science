# Course Modules - Lesson Materials

This directory contains detailed lesson materials for the AI Cloud Enterprises Data Science Learning Platform.

## Structure

Each module has its own directory containing comprehensive README files and lesson materials in Markdown format:

```
modules/
├── module0_foundations/
│   ├── README.md
│   └── lesson_python.md
├── module1_data_analysis/
│   └── README.md
├── module2_machine_learning/
│   └── README.md
├── module3_deep_learning/
│   └── README.md
├── module4_bigdata_mlops/
│   └── README.md
└── module5_professional/
    └── README.md
```

## Available Modules

### Module 0: Foundations
**Directory**: `module0_foundations/`  
**Duration**: 3-4 weeks | 40-50 hours

**Overview**: Establish core Python programming and mathematical skills essential for data science, with Indian industry examples and real-world applications.

**Key Topics**:
- Python for Data Science fundamentals
- NumPy, Pandas, Matplotlib essentials
- Basic mathematics and statistics
- Development environment setup

**Lessons**:
- `README.md` - Complete module overview, topics, case studies, projects, and outcomes
- `lesson_python.md` - Comprehensive Python for Data Science lesson
  - Why Python for data science in India
  - Core concepts with Indian e-commerce examples
  - Pandas/NumPy crash course with local data
  - Mini-assignment on Indian sales data
  - References and resources

**Career Target**: Python Data Analyst (₹4-7 LPA)

## Usage

These lesson materials complement the content available through the platform's backend API (`backend/data/modules.json`). 

- **For Students**: Follow the lessons in order, complete exercises and assignments
- **For Instructors**: Use these as teaching materials and references
- **For Developers**: Integrate these materials into the platform or use as content source

## Format

All lessons are written in Markdown format with:
- Clear section headers
- Code examples with syntax highlighting
- Indian context and examples
- Practical exercises and assignments
- Reference materials and resources
